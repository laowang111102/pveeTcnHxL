# 项目简介 java556

本项目是一款基于 Spring Boot 和 Vue 的协同商品管理系统，采用前后端分离的架构设计。

## 技术栈

本项目采用以下技术构建：

- Java
- Spring Boot
- MySQL
- MyBatis
- Maven
- Vue

## 功能模块

系统包含但不限于以下功能模块：

- 商品管理
- 协同工作流管理
- 用户权限管理

## 系统角色

系统主要角色划分为：

- 系统管理员
- 普通用户

## 运行环境

- JDK 1.8
- Maven
- MySQL 5.7
- Node.js 14 版本

## 部署步骤

1. 准备运行环境
2. 构建后端项目
3. 构建前端项目
4. 部署至服务器

```
├── frontend         前端代码
└── database         数据库文件
```
